# process-analyzer
Analyze a TSV file to find ID with highest consumption between start time and end time
https://github.com/usegutierrez/process-analyzer/blob/master/src/main/resources/screenshot.png
## Getting Started

### Prerequisites
* Java 8
* Maven 3

### Build
```
* mvn clean install -U
```

### Running the App
```
mvn spring-boot:run
```
